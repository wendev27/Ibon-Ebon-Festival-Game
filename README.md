## 📖 Project Overview

Ibon-Ebon Festival Game is an Android educational game that I developed during my **second year at university**.

The application was created as part of our department's **Ibon-Ebon Festival**, where each section designed software inspired by the festival theme.

Players identify the correct bird through interactive gameplay while competing for high scores that are synchronized using **Firebase Realtime Database**.

This repository is preserved as part of my software engineering portfolio to document my early Android development experience and my learning journey.

---

# 🎯 Objectives

The project was designed to:

- Create an interactive educational mobile game
- Teach players about different bird species
- Practice Android game development
- Learn Firebase integration
- Improve Java programming skills
- Explore mobile UI and game mechanics
